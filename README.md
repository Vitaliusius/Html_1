# Новое русское вино

Сайт магазина авторского вина "Новое русское вино".

## Зависимости

Python3 должен быть уже установлен. Затем используйте pip (или pip3, если есть конфликт с Python2) для установки зависимостей:

```
pip install -r requirements.txt
```

## Запуск

Для работы скрипта необходима таблица с напитками Excel хранящаяся в файле с именем 'drinks.xlsx' с названиями  и количеством столбцов таблицы как в примере ниже:


| Категория    |   Название |     Сорт      |   Цена   |     Картинка    |  Акция               |
|--------------|------------|---------------|----------|-----------------|----------------------| 
| Красные вина | Хванчкара  | Александраули |   399    |  hvanchkara.png | Выгодное предложение |
| Белые вина   | Ркацители  | Ркацители     |   399    |  rkaciteli.png  |                      |
| Лимонад      | Буратино   |               |   150    |  buratino.png   | Выгодное предложение |


Пользователю дана возможность указать путь к файлу с таблицей. Для этого необходимо создать файл '.env' в корневом каталоге проекта и присвоить переменной PATH_TABLE путь к файлу с таблицей как в примере ниже (по умолчанию указан корневой каталог проекта):
```
PATH_TABLE = 'C:\python_scripts\wine\drinks.xlsx'
```

Затем запускаеме скрипт командой:

```
python main.py
```

### Проверка работы скрипта

Перейдите на сайт по адресу [http://127.0.0.1:8000](http://127.0.0.1:8000).
Если скрипт удачно запущен, то сайт отобразит страницу магазина с напитками отфильтрованными по категориям.
